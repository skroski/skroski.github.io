Arrays e Iterações

  • .forEach() Maneira tradicional de iterar elementos
  • .map() Transforma e retorna uma nova array com valores atualizados da iteração
  • .reduce() Retorna um acumulador de um processo, reduzindo para o resultado total. Ex: Soma Total ou Maior Valor
  • .reduceRight() Retorna um acumulador de um processo inverso, reduzindo para o resultado total. Percorre a lista toda e acumula também.
  • .some() Retorna true ou false se um dos itens contém na Array de objetos
  • .find() Retorna o valor atual da primeira iteração verdadeira
  • .findIndex() Retorna o posicionamento index da primeira interação
  • .filter() Retorna em uma array todos os objetos que forem verdadeiros com a condição
