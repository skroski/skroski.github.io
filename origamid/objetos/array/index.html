Exercícios de Array JavaScript

  1. .shift() Remove o primeiro valor da array
  2. .pop() Remove o último valor da array
  3. .unshift() Adiciona um item no início da array (neste caso foram 2 itens)
  4. .push() Adiciona um item no array ao final da Array
  5. .sort() Organiza os itens da array em ordem alfabética ou numeral
  6. .reverse() Reverte a ordem dos elementos da Array
  7. .includes() Verfifica se existe um elemento correnspondente na Array e retorna true ou false
  8. .split() Recorta um elemento correspondente.
  9. .join() Conecta um elemento correspondente.
  10. .slice() Cria uma cópia de um novo array a partir do objeto correspondente.
